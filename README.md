# Datové úložiště disertační práce

Tento adresář obsahuje statickou stránku s odkazy na podkladová data k disertační práci. Stránka je určena pro zveřejnění přes GitHub Pages.

## Obsah
- DC1, DC2 a DC3 jsou hlavní sekce datového úložiště.
- Metadata a schéma jsou součástí stránky.
- Jednotlivé soubory ke stažení jsou uloženy přímo v této složce vedle index.html.

## Soubory v adresáři
- schema_VYSLEDKY.jpg je zobrazován v sekci se schématem.
- upol.png je použito jako logo v zápatí.

## Publikace na GitHub Pages
- Při publikaci nastavte GitHub Pages tak, aby směřoval na složku Web.
- Odkazy používají relativní cesty, proto musí být data ve stejné složce jako index.html.

## Údržba
- Pro přidání souboru vložte nový odkaz (tlačítko) do index.html.
- Při změně názvu souboru je nutné upravit i odkaz na stránce.
