# Datové úložiště disertační práce

Tento adresář obsahuje statickou stránku s odkazy na podkladová data k disertační práci: Hodnocení použitelnosti interaktivních kartografických vizualizací metodou eye-tracking.

## Obsah
- DC1, DC2 a DC3 jsou hlavní sekce datového úložiště.
- PS1–PS5 jsou podsekce pro DC3 datového úložiště.
- Metadata a schéma jsou součástí stránky.
- Strukturově jsou jednotlivé soubory ke stažení uloženy přímo v této složce vedle index.html.
